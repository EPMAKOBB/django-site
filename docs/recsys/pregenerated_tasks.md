# Управление предсгенерированными вариантами

## Просмотр и фильтрация

В админке Django появился отдельный раздел **Pre-generated datasets**. В списке отображаются ID, связанное задание, статус активности, параметры и временные метки. Для быстрого поиска используйте фильтры по заданию и признаку активности, а также строку поиска по JSON-параметрам.

Списки автоматически разбиваются на страницы (по 50 записей), поэтому даже большие пулы вариантов загружаются и прокручиваются без проблем. При необходимости можно переходить между страницами через стандартную навигацию админки.

## Переход из карточки задания

На странице редактирования задания в блоке «Предсгенерированные варианты» есть ссылка на соответствующий список с уже применённым фильтром по текущему заданию. Ссылка открывается в новой вкладке, чтобы не потерять несохранённые изменения в форме.

## Редактирование

Встроенный inline-блок теперь отображает существующие варианты только для чтения и не позволяет добавлять новые напрямую из карточки задания. Управляйте вариантами через отдельный раздел — там доступны те же поля и быстрый переход к нужному заданию.
